Role Guidelines

What each editorial role can and cannot do.

SE Senior Editor Scope

✓ All AE capabilities

✓ Review and decide on AE-escalated content

✓ Remove content that clearly violates policy

✓ Issue major revision orders

✓ Monitor SE team performance

✓ Propose policy changes to CE

✗ Cannot approve policies — CE only

✗ Cannot permanently suspend authors — CE only

✗ Ambiguous / novel cases must be escalated to CE

CE Chief Editor Scope

✓ All SE capabilities

✓ Final authority on all escalated decisions

✓ Approve, amend, or reject policy changes

✓ Permanently suspend or reinstate author accounts

✓ Invite, assign, and manage all editorial staff

✓ Set editorial precedent for novel content types

✓ Platform-wide editorial health oversight
